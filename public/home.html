API Reference

Simple REST API for programmatic access.

POST /api/request-otp Request OTP via email
POST /api/verify-otp Verify OTP & create session
POST /api/upload Upload a file
GET /api/files List all files
GET /api/file/:id/versions Get version history
POST /api/file/:id/restore/:version Restore a version
DELETE /api/file/:id Delete a file
